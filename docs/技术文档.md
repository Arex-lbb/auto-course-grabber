## 1. 系统架构

```
┌──────────────────────────────────────────────────┐
│                   Electron Shell                  │
│  ┌────────────┐  ┌──────────┐  ┌──────────────┐ │
│  │  main.js   │  │preload.js│  │  HTTP Server  │ │
│  │ IPC 处理器  │  │ 安全桥接  │  │  :9999 (静态) │ │
│  └─────┬──────┘  └────┬─────┘  └──────┬───────┘ │
│        │              │               │          │
├────────┼──────────────┼───────────────┼──────────┤
│        │     Renderer Process         │          │
│  ┌─────┴──────────────┴───────────────┴────────┐ │
│  │              React App (App.tsx)             │ │
│  │  8 个功能面板 + 抢课引擎 × 3 个策略           │ │
│  └─────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────┘

后端 API 层 (Node.js 主进程, CommonJS):
  src/main/
    request-client.js — Axios 客户端 (Keep-Alive 连接池 + 自动重试)
    course-api.js     — 教务系统 API 封装 (12 个端点)
    auth-store.js     — Token/凭据加密存储 (Electron safeStorage)
    sm2.js            — SM2 国密加密
    jwt.js            — JWT 解析
    term-resolver.js  — 学期 ID 解析
    token-monitor.js  — Token 心跳 + 保活

前端 (React 渲染进程, Vite 构建):
  src/App.tsx                — 根组件 + 全部业务逻辑
  src/App.css                — 暗色主题 (GitHub 风格)
  src/core/grab-engine.ts    — 容量监控抢课 / 分组多班择一
  src/core/realtime-grab.ts  — 实时卡点抢课 (时间校准 + 连接预热)
  src/core/retry-policy.ts   — 重试策略 (指数退避 + 抖动)
  src/shared/types.ts        — 类型定义
  src/shared/constants.ts    — 常量
  src/components/Timetable.tsx — 可视化课表
  src/components/Toast.tsx   — 通知提示
  src/lib/courseDisplay.ts   — 课程时间解析
```

## 7. 网络层优化

| 优化项 | 说明 |
|--------|------|
| HTTP Keep-Alive 连接池 | 模块级 `https.Agent` 单例，`keepAlive: true`，最大 50 并发连接，空闲保留 30s |
| 自动重试拦截器 | Axios 响应拦截器，超时/网络错误/5xx 自动重试 2 次，指数退避 200ms→400ms + 随机抖动 (0-150ms) |
| 超时优化 | 单次超时从 15s 降至 10s，有重试兜底后无需过长等待 |

## 8. 安全措施

| 措施 | 说明 |
|------|------|
| Token 加密存储 | Electron `safeStorage.encryptString()` (OS 级密钥管理) |
| 凭据加密存储 | 学号密码同用 safeStorage 加密 |
| 上下文隔离 | `contextIsolation: true`, `nodeIntegration: false` |
| SM2 加密 | 所有教务 API 请求参数使用国密 SM2 加密 |
| webSecurity 放宽 | `false` — 教师评分模块需跨域 fetch cancir.xyz |

## 9. 已知限制

### 低风险
- **HTTP Server 同步 readFileSync** — 静态文件读取时短暂阻塞事件循环，Electron 单用户场景可忽略
- **时间同步依赖 Date 响应头** — 如果教务服务器未设置 Date 头，回退到本地时间
- **忙等自旋** — `preciseSleep` 最后 50ms 使用 while 忙等，消耗 ~50ms CPU，可忽略
- **syncServerTime 重试重叠** — 时间同步请求也经过 retry 拦截器，每个采样可能重试 2 次，不影响精度

### 中风险
- **申请列表全量加载** — 自动翻页收集 (最多 20 页)，大量申请时加载慢
- **Token 过期不可自动刷新** — 用户需手动重新登录

## 10. 构建与部署

```bash
# 开发
npm run dev          # Vite 开发服务器

# 构建 + 打包
node scripts/pack.js  # npm run pack

release/win32-x64/
  launcher.exe              # 双击启动
```
